# openkaze

> [!NOTE]
> openkaze is currently experimental and not yet usable.

openkaze is a modular traffic orchestrator designed for flexible network composition.

# Anti Goals

openkaze is NOT:
- a monolithic proxy core
- a protocol zoo
- config-driven
- mobile-first (e.g., iOS, Android)

# Stability

openkaze prioritizes architecture correctness, security boundaries, and long-term evolvability over strict backward compatibility.

Compatibility is NOT considered a security boundary.
